Detailed Description

Ibeo FUSION SYSTEM/ECU Set Filter Command.

Before the Ibeo FUSION SYSTEM or ECU sends data after connecting to it (default port 12002), a filter command must be sent.

For sending commands to the ibeo LUX the data type 0x2010 is used. The ibeo LUX replies to a command with a dedicated reply message.

Examples:
IbeoSdkFileDemo.cpp.
