overridevirtual

Checks whether the thread for handling TCP/IP connections to the hardware is running.

Returns
true, if the thread is running, false otherwise.
Note
This should not be mixed up with isConnected(). A device is running if the corresponding thread is running, no matter if the connection to the hardware is established or not.
