Detailed Description

"Earth centered earth fixed" point (aka ECR "earth centered rotational")

Date
Mar 17, 2016

Class to store the ECEF coordinates of a GPS point. For performance reasons it also holds the sine and cosine of the GPS point's latitude and longitude.

Examples:
IbeoSdkPointCloud7510Demo.cpp.
