Detailed Description

A LaneSegmentIn6972 represents a Lane segment within a parent Lane. Each LaneSegment has a unique id within the parent Lane

A Lane Segment ends if the approximation error through straight bounding lines reaches a certain value, or if a CarriageWaySegmentIn6972 ends.

The segmentation of a whole CarriageWayIn6972 is as following:

CarriageWayIn6972 → CarriageWaySegmentIn6972 → LaneIn6972 → LaneSegmentIn6972

The connection is handled using smart pointers. Therefore it is not possible to create an instance of this class, but calling create will return a shared pointer to a new LaneSegment.
