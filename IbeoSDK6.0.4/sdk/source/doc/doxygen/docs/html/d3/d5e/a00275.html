Detailed Description

Specialized data container for images (0x2403)

Ibeo FUSION SYSTEM/ECU image: Video image w/ mounting position and device ID, no detailed description available (generic)

General data type: ibeo::common::sdk::Image

Examples:
IbeoSdkFileDemo.cpp.
