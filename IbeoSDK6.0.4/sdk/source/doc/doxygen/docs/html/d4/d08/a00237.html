Detailed Description

Ibeo frame end separator.

FrameEndSeparator is placed at the end of one frame (e.g. frame like ScanPtList with vehicleData, DeviceStatus and ObjectData).

Examples:
IbeoSdkFileDemo.cpp, IbeoSdkLuxHrLiveDemo.cpp, IbeoSdkMiniLuxLiveDemo.cpp, IbeoSdkScalaLiveDemo.cpp, and ScalaUdpLiveDemo.cpp.
