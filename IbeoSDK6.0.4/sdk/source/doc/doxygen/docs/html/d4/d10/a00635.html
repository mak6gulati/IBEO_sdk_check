Detailed Description

Ibeo FUSION SYSTEM/ECU vehicle state.

Vehicle state data available from Ibeo FUSION SYSTEM and Ibeo AppBase2 (ECU).

All angles, position and distances are given in the ISO 8855 / DIN 70000 coordinate system.

General data type: ibeo::common::sdk::VehicleState

Examples:
IbeoSdkFileDemo.cpp, and IbeoSdkPointCloud7510Demo.cpp.
