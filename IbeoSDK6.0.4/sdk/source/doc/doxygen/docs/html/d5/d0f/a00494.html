Coordinate transformation of a given point in the originRefPlane (originOffset) into the targetRefPlane.

Use this method if you want to transform more than one point into the same target reference plane. Calculate targetRefPointsEcef using PositionWgs84::llaToEcef. This will avoid duplicate calculation other than using transformToShiftedReferencePlane(const ReferencePlane&, const ReferencePlane&, const Vector3<float>&, Vector3<float>&)

Parameters
[in]originRefPlaneThe origin reference plane.
[in]targetRefPointsEcefThe ECEF coordinates and sine and cosine of the latitude and longitude of the target plane reference point.
[in]invTargetRotMatrixInverse rotation between global coordinates and relative coordinates in the target plane.
[in]originOffsetThe point's xyz coordinates in the origin reference plane.
[out]targetOffsetOn exit the point's xyz coordinates in the target reference plane.
Note
F3d stand for full 3D. The altitude will be calculated.

Coordinate transformation of a given point in the originRefPlane (originOffset) into the targetRefPlane.

Use this method if you want to transform more than one point into the same target reference plane. Calculate targetRefPointsEcef using PositionWgs84::llaToEcef. This will avoid duplicate calculation other than using transformToShiftedReferencePlane(const ReferencePlane&, const ReferencePlane&, const Vector3<float>&, Vector3<float>&)

Parameters
[in]originRefPlaneThe origin reference plane.
[in]targetRefPlaneThe target reference plane.
[in]targetRefPointsEcefThe ECEF coordinates and sine and cosine of the latitude and longitude of the target plane reference point.
[in]invTargetRotMatrixInverse rotation between global coordinates and relative coordinates in the target plane.
[in]originOffsetThe point's xyz coordinates in the origin reference plane.
[out]targetOffsetOn exit the point's xyz coordinates in the target reference plane.
Note
P3d stand for partial 3D, altitude will taken from the point given by originOffset in originRefPlane. If this is not set, the altitude of the originRefPlane GPS anchor point will be used.
