Detailed Description

GpsImu.

This data type contains unprocessed position and motion data provided by a connected GPS/IMU device, e.g. XSens MTi-G.

General data type: ibeo::common::sdk::GpsImu

Examples:
IbeoSdkFileDemo.cpp.
