using ibeo::common::sdk::CommandCBase::CommonBase = CommandCBase

Used by templates to identify this class when receiving derived classes.

CommandCBase alias CommonBase is the class to be shared by all derived special command classes. The CommonBase alias is used in templates as BufferAndImporterProvider to identify the correct map.
