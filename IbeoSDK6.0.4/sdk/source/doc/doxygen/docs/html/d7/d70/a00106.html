Detailed Description

An content entry in the variable part of this data type.

Date
Jul 30, 2013

An object of this class describe a content entry in the variable part of this data type. Apart from the ContentId, the ElementType and the number of used bytes it also contains a pointer to the content which is located in the content buffer of the DeviceStatus6303 object.
