Detailed Description

List of objects in the format of the LUX3 firmware.

Object data available from ibeo LUX laserscanners (not available for ibeo LUX prototypes). Each data block starts with a header followed by the object list. Each object has a list of contour points. The Sigma values are calculated by Kalman filter by taking into account the object age.

Examples:
IbeoSdkFileDemo.cpp, IbeoSdkLuxLiveDemo.cpp, IbeoSdkScalaLiveDemo.cpp, IbeoSdkTrackingBoxLiveDemo.cpp, and ScalaUdpLiveDemo.cpp.
