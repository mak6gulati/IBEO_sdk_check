Detailed Description

General data type: ibeo::common::sdk::Scan

Examples:
CommaSeparatedValuesWriterDemo.cpp, and IbeoSdkFileDemo.cpp.

VehicleCoordinates 

Bit 11: scan point coordinate system: 0 = scanner coordinates, 1 = vehicle / reference coordinates.

ObjIdAsSegId 

Bit 12: segment ID of scan points is used as object ID.

RayAngleCorrected 

Bit 13: ray angle corrected (window refraction, etc.)

void ibeo::common::sdk::Scan2209::clearLabelFlags ( )
inline

Removes all processing status flags ("...labeled") from the Scan object, but leaves the other flags unchanged.

Note: This only changes the flags of the Scan object, not the flags in the individual scan points! The flags in the individual points are unchanged and must be modified in a manual iteration loop.
