Member Enumeration Documentation

enum ibeo::common::sdk::TcpReceiveThreadEnsemble::ReconnectMode : uint8_t
strong
Enumerator
InitialConnect 

Immediately connect to the remote device after calling getConnected(). If failed or if the connection gets lost afterwards, it will not be re-established (legacy behaviour).

WaitForRemoteDevice 

Upon calling getConnected(), wait until the remote device is available, stop if the connection gets lost afterwards.

AutoReconnect 

Like WaitForRemoteDevice, but try to reconnect every time the remote device is disconnected.
