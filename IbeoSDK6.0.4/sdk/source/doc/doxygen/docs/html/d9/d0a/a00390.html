Detailed Description

Ibeo FUSION SYSTEM/ECU Trace Messages.

Software modules that are deploying the Ibeo FUSION SYSTEM for communication can sent trace messages consisting of a character string. Trace Messages are distributed with four different data types dependent on their priority

Examples:
IbeoSdkFileDemo.cpp, IbeoSdkLuxLiveDemo.cpp, IbeoSdkScalaLiveDemo.cpp, IbeoSdkTrackingBoxLiveDemo.cpp, and ScalaUdpLiveDemo.cpp.
