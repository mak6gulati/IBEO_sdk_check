Detailed Description

Current draft of device status definition for Scala B2 >X100 (generic)

General data type: ibeo::common::sdk::DeviceStatus

Examples:
IbeoSdkFileDemo.cpp, IbeoSdkScalaLiveDemo.cpp, and ScalaUdpLiveDemo.cpp.
