Detailed Description

template<typename TLaneSegment>
class ibeo::common::sdk::lanes::CarriageWaySegmentTemplate< TLaneSegment >

A CarriageWaySegmentTemplate represents a single segment of a CarriageWayTemplate.

Each CarriageWaySegmentTemplate has a unique id within the parent CarriageWayTemplate.

A CarriageWayTemplate holds a constant number of lanes of type LaneTemplate. The segmentation of a whole CarriageWayTemplate is as following:

CarriageWayTemplate → CarriageWaySegmentTemplate → LaneTemplate → LaneSegmentInTemplate

The connection is handled using smart pointers. Therefore it is not possible to create an instance of this class, but calling create() will return a shared pointer to a new CarriageWaySegmentTemplate.
