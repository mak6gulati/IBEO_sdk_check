Detailed Description

Date
Jul 12, 2016

Class to receive ibeo messages from an TCP/IP socket and prepare them for decoding.

[out]startOfMsgBodyIf a complete message is in the buffer, startOfMsgBody will point to the start of the message body inside the objects buffer. Otherwise startOfMsgBody will be NULL.
Returns
true if a complete message is in the buffer, ready to be processed. The message is described by recvDataHeader and startOfMsgBody. false There is no complete message in the buffer yet. recvDataHeader and startOfMsgBody will be NULL.
