bool ibeo::common::sdk::IdcFileController::setToFirstDataBlock ( const bool  clearStreamFlags = false)

Move the file pointer in front of the first DataBlock.

The file pointer is moved in front of the first message, the IbeoDataHeader is read and the file pointer is moved back in front of the first message.

Parameters
[in]clearStreamFlagsIf true, the stream flags will be cleared before the operation will be performed.
Returns
true if the operation was successful, i.e. the stream was set in front of the first DataBlock and at this position a IbeoDataHeader could be read.
