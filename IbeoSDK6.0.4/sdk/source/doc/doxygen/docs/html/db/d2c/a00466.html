enum ibeo::common::sdk::ObjectIn2280::Flags : uint16_t
strong
Enumerator
TrackedByStationaryModel 

is object tracked using stationary model

Mobile 

Has been detected/validated as mobile. (the current tracking model is irrelevant; this flag just means it has been moving at some time)

Validated 

Object (stationary or dynamic) has been validated, i.e. valid enough to send out to the interface.
