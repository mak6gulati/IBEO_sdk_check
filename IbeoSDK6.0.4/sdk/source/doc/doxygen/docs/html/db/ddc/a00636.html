int ibeo::common::sdk::Version::getVersion ( )

Get the version as an integer.

The value will be calulated as MMmmRRPP, where MM is the major version, mm the minor version, RR the revision number and PP the patch level number.

A correct output will only be produced if all of these figures are not larger than 99.

Note
build and info will not be used.
std::string ibeo::common::sdk::Version::toString ( ) const

Return the version as a string.

The format is MM.mm.RR.PP (Build BB) (info). Where MM is the major version, mm the minor version, RR the revision number, PP the patch level number, BB the build number and info the info string.

Returns
The version as a string.
