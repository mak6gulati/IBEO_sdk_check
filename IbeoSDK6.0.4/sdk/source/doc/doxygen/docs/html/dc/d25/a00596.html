Detailed Description

SystemMonitoringDeviceStatus.

The SystemMonitoringDeviceStatus data type provides information about connected devices: Scanner, Cameras, GPS, IMU. If devices have lost connection to the perception ECU, this datatype is also used to signal the loss of these devices.
