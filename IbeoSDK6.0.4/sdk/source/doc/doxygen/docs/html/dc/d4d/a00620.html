Detailed Description

template<typename T>
class ibeo::common::sdk::TransformationMatrix3d< T >

Transformationmatrix class which holds a 3x3 rotation matrix and a 3x1 dimensional position vector. The size of the matrix is 4x4. The structure is:

| rotM posV | | 0 0 0 1 |

Dedicated to be used for 3d coordinate transformations
