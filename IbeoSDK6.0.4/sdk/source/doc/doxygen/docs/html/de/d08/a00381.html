Detailed Description

A LaneSegmentIn6970 represents a Lane segment within a parent Lane. Each LaneSegment has a unique id within the parent Lane

A Lane Segment ends if the approximation error through straight bounding lines reaches a certain value, or if a CarriageWaySegmentIn6970 ends.

The segmentation of a whole CarriageWayIn6970 is as following:

CarriageWayIn6970 → CarriageWaySegmentIn6970 → LaneIn6970 → LaneSegmentIn6970

The connection is handled using smart pointers. Therefore it is not possible to create an instance of this class, but calling create will return a shared pointer to a new LaneSegment.
