Detailed Description

TimeRecord.

This data type contains the local system time as well as a time signal that is provided by an external device, e.g. a GPS receiver connected to the Ibeo ECU. This data can be used for offline transformation of the time base of the recorded data stream.

Examples:
IbeoSdkFileDemo.cpp.
