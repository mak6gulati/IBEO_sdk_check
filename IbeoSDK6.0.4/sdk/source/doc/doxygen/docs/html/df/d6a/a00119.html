Detailed Description

Abstract base class for all listener.

A DataContainerListener can be registered to an IbeoDevice to receive all DataBlock of the associated DataType received by that device.

Derived classes will have to implement an onData method for the associated DataType. This onData method will be called in the context of the receive thread of that device.

The data received by the onData method will be deserialized.

In case one is not interested in the contents of that DataBlock it may be better to implement a DataStreamer.
